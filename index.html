Mis trabajos

SpeedService

Este trabajo nació durante la cursada de Laboratorio IV, al momento de buscar una idea de proyecto para desarrollar el examen final de dicha materia. SpeedService es una aplicación web que permite ofrecer, mostrar o solicitar servicios de remis, mandado y flete en una ciudad determinada. El objetivo principal fue crear una plataforma flexible, adaptable a otros servicios, que oficie de nexo entre proveedores y potenciales clientes con el fin de satisfacer las necesidades de estos últimos de una manera cómoda y eficiente.

Biblioteca escolar - DB

El objetivo de este trabajo fue aplicar los conocimientos adquiridos en la materia Diseño y Administración de Base de Datos para poder elaborar las herramientas necesarias que permitan analizar, gestionar y administrar los datos de un sistema perteneciente a una biblioteca escolar.

API REST

Este trabajo consistió en realizar una API REST con NodeJS, un archivo HTML que envíe una petición a la API para que ejecute una transacción (ABM) en una cierta base de datos y otro archivo HTML para mostrar los datos resultantes.
